# Documentação do Projeto ConnectDB

Bem-vindo à documentação oficial do projeto **ConnectDB**. Este projeto é uma aplicação fullstack moderna que utiliza as melhores práticas de desenvolvimento para gerenciar usuários e produtos.

## Estrutura da Documentação

Para facilitar o entendimento, dividimos a documentação nos seguintes tópicos:

1.  **[Arquitetura Backend](backend.md)**: Detalhes sobre NestJS, Prisma, PostgreSQL e a estrutura da API.
2.  **[Arquitetura Frontend](frontend.md)**: Detalhes sobre React, Vite, Axios e a interface do usuário.
3.  **[Guia do Swagger](swagger-guide.md)**: Como o Swagger foi implementado e como utilizá-lo para testar a API.

## Visão Geral do Projeto

O **ConnectDB** foi desenvolvido para ser uma base sólida para aplicações que precisam de autenticação, gerenciamento de dados e uma interface responsiva e elegante.

### Tecnologias Principais

- **Backend**: NestJS (Node.js framework)
- **Frontend**: React.js com Vite
- **Banco de Dados**: PostgreSQL (rodando em Docker)
- **ORM**: Prisma
- **Documentação de API**: Swagger/OpenAPI
- **Estilização**: CSS Moderno (Glassmorphism)
